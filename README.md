# xrandr-watcher
xrandr event watcher

## Requirements

* `nodejs` (I use `12.10.0` during development)

## build

```bash
git clone git@github.com:pastleo/xrandr-watcher.git
cd xrandr-watcher
make
```

then `./xrandr-watcher` is the built executable

## development

```bash
git clone git@github.com:pastleo/xrandr-watcher.git
cd xrandr-watcher
npm install
npm start
```
